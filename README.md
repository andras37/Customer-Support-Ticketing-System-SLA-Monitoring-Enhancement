This repository contains a Business Analyst portfolio project focused on enhancing SLA monitoring capabilities within an existing customer support ticketing system.

## Project Overview
The project addresses a common operational challenge: lack of real-time visibility into SLA performance during ticket handling.  
The proposed enhancement enables proactive monitoring, escalation, and improved operational control.

## Business Objective
- Improve visibility of SLA status at ticket level
- Enable early intervention before SLA breaches
- Reduce reactive escalations and customer dissatisfaction
- Support operational decision-making for support managers
